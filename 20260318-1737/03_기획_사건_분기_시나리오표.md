# 1. 시점별 핵심 이벤트 및 분기 (타임라인)

## [Day 1] 저녁 ~ 심야
* **[메인 이벤트]** 아서 블랙우드의 유언장 전면 수정 선언 후 만찬, 그리고 서재에서 독살됨.
* **[분기 1: 사건 직후의 행동]** (행동 기회 1회)
  * **A. 서재에 남아 현장을 지킨다.** -> 메리와 대화. 메리의 신뢰도 증가. `단서: 서재 안쪽에서 잠긴 문` 확인.
  * **B. 복도를 수색한다.** -> 리처드를 목격. 리처드의 의심도 증가. **위험도(Danger) 1 상승**.
  * **C. 온실로 향하는 누군가를 쫓는다.** -> 폭풍우 때문에 발자국만 발견. 에블린에 대한 초기 의심 플래그 획득.

## [Day 2] 오전 ~ 오후 (본격적인 추리)
* **[메인 이벤트]** 경찰은 다리가 끊겨 3일째 오전에야 도착한다는 무전 연결. 남은 사람들의 알리바이 조사 시작.
* **[분기 2: 알리바이 및 방 수색]** (행동 기회 2회)
  * **A. 아서의 침실 수색** -> `단서: 매일 복용하던 심장약 병` 획득. (진엔딩 필수)
  * **B. 온실(아틀리에) 조사** -> 토마스가 막아섬. (토마스 신뢰도가 낮으면 진입 불가, 억지로 진입 시 **위험도 1 상승** 및 평판 하락)
  * **C. 리처드 추궁** -> 리처드가 도박 빚이 있다는 사실 획득. 리처드는 자신을 변호하며 "에블린이 차를 타는 것을 봤다"는 거짓/진실 섞인 힌트 제공.
  * **D. 오펜하이머 박사 추궁** -> 아서가 길어야 3개월 시한부였다는 `단서: 비밀 의료 기록` 획득. (동기 추론의 핵심)

## [Day 2] 심야 (생존의 기로)
* **[분기 3: 심야의 선택 - 가장 위험한 시간대]**
  * **A. 온실 몰래 잠입** -> `단서: 입양 기록서 및 가문 횡령 증거` 획득. 단, **현재 누적 위험도가 2 이상일 경우 에블린에게 발각되어 살해당함 (배드엔딩 2)**.
  * **B. 자신의 방에 머무름** -> 안전 확보. 하지만 온실의 결정적 증거를 놓쳐 진엔딩 불가 (노멀엔딩이나 배드엔딩 1로 직행).
  * **C. 토마스와 접선** -> 이전 분기에서 메리 신뢰도를 올려둔 경우 가능. 토마스를 설득하여 에블린의 비밀문에 대해 털어놓게 만듦. (안전하게 정보 획득)

## [Day 3] 아침 (결단의 시간)
* **[메인 이벤트]** 폭풍우가 잦아들고 경찰차가 멀리서 접근함. 최후의 범인 지목.
* **[분기 4: 범인 지목]**
  * **A. 에블린 지목** + `심장약` + `입양 기록서` 제시 -> 완벽한 논리로 제압. 에블린 자백. **(최종 진엔딩: 진실의 폭로)**
  * **B. 에블린 지목** + 증거 부족 -> 에블린이 역공. 동기를 입증하지 못해 에블린이 토마스를 희생양으로 몰고 자신은 요트를 타 도망침. **(노멀엔딩: 공범의 도주)**
  * **C. 리처드 지목** -> 도박 빚 동기로 몰아붙임. 진짜 범인인 에블린이 맞장구를 치며 리처드가 체포됨. 사건 미궁. **(배드엔딩 1: 억울한 희생양)**
  * **D. 지목 포기 / 시간 초과** -> 경찰 진입. 치열한 눈치싸움만 하다 미제 사건으로 종결. **(미해결 엔딩: 침묵의 저택)**

# 2. 내부 관리 파라미터 작동 로직 (Flutter State 관리용)

이 게임의 분기 시스템은 아래 **4가지 핵심 Data State**에 의해 결정됩니다.

- **시간 (Time)**: 행동을 할 때마다 1씩 소모. 정해진 시간 내에 단서를 찾아야 함.
- **단서 (Evidence)**: Set 또는 List 형태로 보관. 특정 단서가 있어야만 선택지가 열리는 조건문 구현. (예: `if (evidence.contains('heart_pills'))`)
- **위험도 (Danger Level)**: Int 값. 특정 NPC를 과도하게 압박하거나 위험한 곳을 혼자 수색하면 증가. **이 수치가 높아지면 사망 분기(즉사기) 발동.**
- **신뢰도 (Trust Map)**: Map<String, int> 형태. `{ '메리': 50, '토마스': 20 }`. 신뢰도가 일정 수치 이상이어야만 안전한 선택지(예: 토마스의 협력)가 해금됨.
