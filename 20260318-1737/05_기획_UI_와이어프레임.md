# UI 와이어프레임 및 무드 보드

추리 어드벤처 특성상 화려한 3D나 액션보다, **분위기 있는 배경 일러스트 + 가독성 높은 텍스트 + 긴장감을 주는 선택지 UI**가 핵심입니다. AI로 생성한 컨셉 이미지를 함께 첨부합니다.

## 1. 메인 타이틀 화면

* **배경**: 절벽 위의 음산한 고딕 양식 저택, 간헐적으로 번개가 치는 애니메이션 에셋 적용.
* **UI 요소**:
  * 중앙 하단: `사건 조사 시작 (새 게임)`
  * `진행 중인 기록 (이어하기)`
  * `수집된 엔딩 (컬렉션)`

## 2. 스토리 진행 화면 (핵심 UI)
![서재 현장](file:///C:/Users/WH_87/.gemini/antigravity/brain/f78c7772-3154-4de8-a23f-d4076c91b484/mansion_study_room_1773823717210.png)
* **상단 (Status Bar)**: 
  * 현재 챕터 / 남은 시간 / (숨겨진 위험도를 암시하는 붉은 오라 연출 등)
* **중앙 (Visual)**: 
  * 투명도 있는 짙은 그라데이션 위에 현재 화자(Speaker)의 초상화 또는 일러스트 배치.
* **하단 (Text Box)**:
  * 타이프라이터 효과로 텍스트 출력.
  * 텍스트 출력이 끝나면 하단에서 부드럽게 선택지(Button) 2~3개가 페이드인(Fade-in) 됨.

## 3. 캐릭터 조우 및 인물 도감 화면
![에블린 컨셉](file:///C:/Users/WH_87/.gemini/antigravity/brain/f78c7772-3154-4de8-a23f-d4076c91b484/evelyn_portrait_1773823911293.png)
* **캐릭터 팝업**:
  * 특정 인물과 대화 시 중앙에 인물 초상화 노출 (예: 에블린).
* **도감(수첩) 뷰**:
  * 화면을 스와이프하거나 우측 상단 수첩 아이콘을 누르면 BottomSheet 형태로 등장.
  * 탭 1: `단서 목록 (Evidence)` - 돋보기 아이콘, 습득 물품 보관.
  * 탭 2: `인물 프로필 (Trust)` - 획득한 인물 정보 및 알리바이 메모.
